### ***Команда cat***

Это базовый инструмент для чтения, создания и объединения текстовых файлов, выводящий их содержимое в стандартный поток вывода (на экран). Она часто используется для быстрого просмотра небольших файлов, объединения нескольких файлов в один или перенаправления данных через конвейер (pipe).

### ***Синтаксис***

> `cat` **опция** ***файл***

### ***Основные способы использования***

1. **Просмотр файла:**

- `cat file.txt` — выводит содержимое file.txt.

2. **Просмотр нескольких файлов:**
    
- `cat file1.txt file2.txt` — отображает их последовательно.

3. **Создание файла:**

- `cat > new_file.txt` — позволяет ввести текст и сохранить его (нажмите Ctrl+D для выхода).

4. **Объединение файлов:**
    
- `cat file1.txt file2.txt > combined.txt` — объединяет file1.txt и file2.txt в combined.txt.

5. **Нумерация строк:**
    
- `cat -n file.txt` — выводит файл с номерами всех строк.

6. **Нумерация только непустых строк:**

- `cat -b file.txt` - выводит файл с номерами непустых строк

7. **Подавление пустых строк:**

- `cat -s file.txt` - cвернуть несколько пустых строк в одну. Две и более пустых строк, идущих подряд, заменяются одной пустой строкой.


### ***Полезные опции:***

- `-n` — нумеровать все строки.
- `-b` — нумеровать только непустые строки.
- `-s` — убирать повторяющиеся пустые строки.
- `-E` — показывать $ в конце каждой строки.
- `-T` — показывать табуляцию как ^I.
