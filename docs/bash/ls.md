### Команда ls

Эта команда используется для вывода списка файлов и каталогов в указанной директории, либо в текущей, если аргументы не заданы; она позволяет просматривать содержимое папок, а с помощью опций (ключей) можно настроить вывод.

### Синтаксис

`ls` **опция** *директория*

### Основные опции

- `-l` - Показать длинный (detailed) список с правами доступа, владельцем, размером и датой изменения. 
- `-a` - Показать все файлы, включая скрытые (начинающиеся с точки . ). 
- `-la (или -al)` - Комбинация, показывающая длинный список всех файлов, включая скрытые. 
- `-h` - Использовать с `-l` (например, `-lh`) для читабельных размеров файлов (КБ, МБ, ГБ). 
- `-R` - Рекурсивный вывод, показывающий содержимое подкаталогов. 
- `-t` - Сортировать по времени модификации (новые файлы первыми). 
- `-S` - Сортировать по размеру (от большего к меньшему). 
- `-r` - Обратная сортировка (например, `-ltr` покажет самые старые файлы в конце длинного списка). 
- `-F` - Добавляет к именам символы, обозначающие тип ( `/` для каталога, `*` для исполняемого файла).
